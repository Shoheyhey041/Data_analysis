# Data_analysis
This directory contains the dataset (imaginative data at a gym) and the analysis of the dataset  

Dataset.xlsx: Excel file containing the data (gym usage, body log, purchase history etc)  
Analysis.ipynb: Analysis of the dataset and some predictive models 
### Predictive models
1: a prediction model whether a user would keep the membership 30 days after the registration, given the data

2: a prediction model how much a user will spend within 30 days after the registration, given the data
