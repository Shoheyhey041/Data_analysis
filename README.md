# Data_analysis
This directory contains datasets and jupyter notebook showing the analysis process of each dataset.

## gym_membership_model
This directory contains the dataset (imaginative data at a gym) and the analysis of the dataset  
Dataset.xlsx: Excel file containing the data (gym usage, body log, purchase history etc)  
Analysis.ipynb: Analysis of the dataset and some predictive models 
### Predictive models
1: a prediction model whether a user would keep the membership 30 days after the registration, given the data  
2: a prediction model how much a user will spend within 30 days after the registration, given the data

## loan_approval
This directory contains the dataset (Kaggle:https://www.kaggle.com/datasets/architsharma01/loan-approval-prediction-dataset) and the analysis of the dataset  
loan_approval_dataset.xlsx: Excel file containing the data (loan approval status and other details which could impact the loan approval)  
Analysis_loan.ipynb: Analysis of the dataset and predictive models of loan approvals 
### Predictive models
1: random forest model   
2: ANN model
